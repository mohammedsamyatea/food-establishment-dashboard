# Food Establishment Dashboard

This project contains a static HTML/CSS mockup of the Food Establishment Profile Dashboard.

## Features
- High risk activity badge
- Last inspection grade badge
- Summary cards
- License information
- Activities & risk assessment
- Inspection history

## How to Deploy on GitHub Pages

1. Create a GitHub account.
2. Create a new repository (public).
3. Upload:
   - index.html
   - styles.css
   - README.md
4. Go to **Settings → Pages**.
5. Under “Build and Deployment”, choose:
   - Source: **Deploy from branch**
   - Branch: **main**
   - Folder: **root**
6. Save.
